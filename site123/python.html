Python

Язык программирования Python был создан в 1989–1991 годах голландским программистом Гвидо ван Россумом. Изначально это был любительский проект, который Гвидо начал, чтобы занять себя на рождественских каникулах. Идея создания нового языка появилась у него двумя годами ранее.

История развития Python включает несколько этапов:

  • В 1991 году Гвидо опубликовал первую версию (0.9.0) языка, включающую базовые возможности.
  • Через три года вышла версия 1.0, в которой функционал был дополнен обработкой списков данных.
  • В 2000 году была опубликована версия 2.0, которая отличалась исправленными недочётами прежних версий, а также новыми полезными функциями для программистов.
  • В 2008 году представлена версия Python 3, включившая возможность печати, поддержку деления чисел и расширенное исправление ошибок.

    • Сегодня Python развивается и поддерживается командой разработчиков. В 2001 году они объединились в некоммерческую организацию Python Software Foundation.

    Кто создал Python

    Создатель языка Python — нидерландский программист Гвидо ван Россум. Он был участником проекта по написанию ABC, языка для обучения программированию. В конце 1989 года Гвидо приступил к разработке нового языка и задумал его как потомка ABC, способного к обработке исключений и взаимодействию с операционной системой Amoeba. Так и получился Python.
