Java

История создания Java началась в 1991 году, когда группа разработчиков компании Sun Microsystems под руководством Джеймса Гослинга приступила к созданию языка Oak («Дуб») для использования в цифровых бытовых устройствах.

Oak должен был стать альтернативой C/C++ как более простой и кроссплатформенный язык. Поскольку язык программирования с таким названием уже существовал, в 1995-м его переименовали в Java — в честь одной из марок кофе.

23 мая 1995 года — дата официального выпуска Java.

Java получил широкую известность благодаря включению виртуальной машины Java Virtual Machine (JVM) в культовый браузер Netscape.

Кто создал Java

Java был создан Джеймсом Гослингом в 1995году, когда он работал в Sun Microsystems. Разработка Java началась в 1991 г., а первая общедоступная версия была выпущена в январе 1996 г. Да, Java — это язык программирования на котором создаются приложения, ОС для смартфонов, корпоративное программное обеспечение и многие другие известные программы.
